## Synopsis

A simple app to allow users to enter search wikipedia pages. Maximum 10 results will be shown with links for user to click on.

## Motivation
A small and simple project to demo vanilla javascript MVC.

## Installation

npm install<br/>
bower install<br/>

## To Run
gulp serve
